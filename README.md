# 日本株投資メモ(jpstockmemo)

### 基本機能

- 銘柄名
- 証券コード
- 市場(東証１部,東証２部,JASDAQ,マザーズ,その他)
- メモ

上記４項目を新規保存、編集、削除ができます。

---
### 今後の機能追加・変更予定

- 一覧表示の順序をソートできるようにする。
- 検索できるようにする。
- ボトムナビゲーションバー(BottomNavigationBar)を追加する。<br>(ページが変更される際の読込データ量をproviderを使用して最小限にするため。)
